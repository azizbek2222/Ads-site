Developer Documentation

SDK Version v1.0.2

1 SDK Integration

Reklamalarni ko'rsatish uchun quyidagi scriptni saytingizning <head> qismiga joylashtiring.

<script type="module" src="https://adsero.vercel.app/sdk.js" data-app-id="YOUR_APP_ID"></script>

2 Reklamani chaqirish

Reklamani ko'rsatish uchun AdseroSDK klassidan foydalaning. Funksiya reklamani tizim tomonidan belgilangan vaqt bilan chiqaradi.

// SDK ni ishga tushirish const adsero = new window.AdseroSDK(); // Reklamani ko'rsatish adsero.showInterstitial();
Xavfsizlik: Reklama vaqti SDK ichida qat'iy belgilangan va foydalanuvchi tomonidan o'zgartirilishi mumkin emas.

To'liq misol (Full Example)

Tugma bosilganda reklamani chiqarish namunasi:

<button onclick="startAd()">Bonus olish</button> <script> function startAd() { if(window.AdseroSDK) { const ad = new AdseroSDK(); ad.showInterstitial(); } else { console.error("SDK yuklanmagan!"); } } </script>
